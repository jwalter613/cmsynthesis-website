Execution

Build sequence

  1. Power only bring-up Verify MB102 rails and polarity first. Do not connect Teensy/audio modules until rails are confirmed.
  2. Stage 1 checklist Run the beginner checklist and multimeter drill in `hardware/breadboard/` before adding logic/audio wiring.
  3. Header prep Solder required pin headers to Teensy 4.1 and Audio Shield for stable breadboard insertion.
  4. Firmware sanity Flash basic test sketches first (`00_audio_sanity`, `01_oled_sh1106_sanity`, `02_midi_in_monitor`).
  5. Audio path test Confirm audible output from demo sketches before moving to unified firmware.
  6. Unified v1 pass Flash `08_cms_alpine_v1`, then validate controls/display/clock incrementally.
